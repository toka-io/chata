

# Chata

## What is Chata?
Chata is a simple chat server solution. 

The chat server can run on any available port and supports the following events:

### Server Events
* activeViewerCount
* disconnect
* join
* sendMessage
* users

### Client Events
* activeViewerCount
* history
* receiveMessage
* users
